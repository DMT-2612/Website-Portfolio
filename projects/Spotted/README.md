# Spotted: Phishing Recognition and Response Prototype

Spotted is a Figma mobile prototype designed to help students and recent graduates practise phishing recognition and response in job-search scenarios.

## Prototype

View the interactive Figma prototype: [Spotted Figma Prototype](https://www.figma.com/proto/CZ2ae9esX4Ah3MsKvY7ye5/Spotted?node-id=0-1&t=oE6XwpxjQfjuhd4f-1)

## Case Study

Read the BA case study: [spotted-case-study.md](spotted-case-study.md)

## Project Context

This project was created for ICT20025 User Experience Design. The focus was on designing an interactive prototype that supports decision-making under uncertainty, feedback, and learning.
