Yarn Petals Requirements Document

Business problem, scope, requirements, business rules, and expected value.

1. Project Context

Yarn Petals is an e-commerce crochet flower brand that needed a structured sales reporting solution during an ownership transition. The business had sales records across Shopify, manual order files, and social media orders, but the files were inconsistent and difficult to reconcile.

The project goal was to create a repeatable workflow that could clean, standardise, and map order data into one universal reporting dataset, then use that dataset in Power BI.

2. Business Problem

  • Shopify exports were messy and not aligned with the structured reporting file.
  • Manual 2025 order data did not follow the same format as other datasets.
  • Instagram and Facebook orders lacked consistent order numbers.
  • Inconsistent categories created risk in sales, product, customer, and channel analysis.
  • The new owner needed a reliable reporting process that did not depend on manual fixes.

3. Project Objectives

  1. Create a universal structured dataset format.
  2. Develop an AI-assisted Python workflow to clean and map Shopify exports.
  3. Standardise manual and social media orders.
  4. Build an interactive Power BI dashboard.
  5. Make the solution transferable and scalable for the owner.

4. Scope

In Scope

  • Analyse existing datasets and define a universal schema.
  • Clean and map Shopify raw exports into the structured format.
  • Standardise manual and social media orders.
  • Generate missing order numbers where required.
  • Build a Power BI dashboard with KPIs, filters, trends, and category breakdowns.
  • Apply Yarn Petals branding and owner-friendly dashboard layout.

Out of Scope

  • Shopify system restructuring.
  • ERP integration.
  • Real-time API integration.
  • Automated Instagram or Facebook order capture.
  • Financial accounting reconciliation.

7. Business Rules

The project required explicit category mapping so the owner could rely on stable reporting categories after handover.

  • Blank product, animal, and colour values are standardised to `NG`.
  • Special customised high-price bundles are mapped to `Largest Bundle`.
  • Bud Bundle style variants are consolidated into `Bud Bundle`.
  • Selected product names drive animal classification for Katsu-themed bundles.
  • Graduation and birthday accessories drive the Occasion field.
  • Selected gift bundle products can be mapped to couple-related occasions when no occasion is provided.

8. Expected Results

  • A single structured dataset as the reporting source of truth.
  • A repeatable cleaning and mapping process for Shopify and manual orders.
  • Reduced data mismatch and manual cleaning effort.
  • Improved reporting accuracy and decision-making confidence.
  • A professional Power BI dashboard ready for owner handover.
